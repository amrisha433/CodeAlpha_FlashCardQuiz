# 📚 Flashcard Quiz App

A clean and intuitive Android flashcard application built with **Jetpack Compose** and **Kotlin**, designed to help users study and memorize concepts efficiently.

---

## ✨ Features

- 📖 **Flashcard Study Mode** — Each card displays a question with a toggleable answer
- ➡️ **Navigation** — Easily move between cards using Next and Previous buttons
- ➕ **Add Flashcards** — Create custom flashcards with your own questions and answers
- ✏️ **Edit Flashcards** — Update any existing flashcard anytime
- 🗑️ **Delete Flashcards** — Remove flashcards you no longer need
- 📊 **Dynamic Progress Counter** — Shows "Card X of Y" that updates in real time
- 🎉 **Completion Screen** — Celebrates when you finish all flashcards
- 🔄 **Study Again** — Restart the deck anytime from the completion screen

## 🚀 Getting Started

### Prerequisites
- Android Studio Hedgehog or later
- Android SDK 24+
- Kotlin 1.9+

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/flashcard-quiz-app.git
```

2. **Open in Android Studio**
```
File → Open → Select the project folder
```

3. **Run the app**
```
Click the Run button or press Shift+F10
```

---

## 📋 How It Works

1. Launch the app and tap **"Start Learning"**
2. Read the question on each flashcard
3. Tap **"Show Answer"** to reveal the answer
4. Navigate using **Next** / **Previous** buttons
5. Use **Add**, **Edit**, or **Delete** to manage your flashcards
6. Tap **Finish** when done — see the **Well Done** screen!
7. Tap **"Study Again"** to restart
